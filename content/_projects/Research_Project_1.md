<h4>Project Overview</h4>

This project focuses on secure and low-latency authentication across distributed
IoT edge environments. Blockchain-based coordination and cryptographic
accumulators are used so authentication data can be shared and verified
efficiently across edge nodes.

**More information:** Visit [https://issedgeauthentication.wordpress.com/](https://issedgeauthentication.wordpress.com/)

In this project, we focus on:

- Secure edge authentication
- Blockchain coordination for trust
- Efficient verification with cryptographic accumulators
