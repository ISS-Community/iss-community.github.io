<h4>Project Overview</h4>

This project addresses operational visibility and resilience in modern data
centers. Flowzen provides real-time traffic monitoring while Iris handles
adaptive collection pipelines to support faster detection and better response in
cloud-scale environments.

**More information:** Visit [https://issdatacentersecurity.wordpress.com/](https://issdatacentersecurity.wordpress.com/)

In this project, we focus on:

- Real-time network traffic monitoring
- Adaptive telemetry collection
- Resilient cloud infrastructure operations
- Security analytics for data center environments
